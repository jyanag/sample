# sample（JavaScript エコシステム入門）

JavaScript とその主要な技術（React, Next.js, Node.js）の役割と関係性を、初心者向けに図形（ベン図）を用いてわかりやすく解説するシングルページWebアプリケーションです。

## 🚀 プロジェクトについて

このリポジトリには、以下のファイルが含まれています。

*   **`index.html`**: CSSによる動的な図形（インタラクティブなベン図）と解説カードを含んだ、ブラウザで動く学習用のWebページです。
*   **`README.md`**: このドキュメントです。
